Summary: Lattitude Vs

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish the analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. lattitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity, as well as explanations and descriptions of any trends and correlations witnessed
